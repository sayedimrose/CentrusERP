# Centrus ERP: Core Database Tables

This database handles all business logic, transactions, master data, and operational modules.

## System & Configuration
- **Centrus_Core_Companies**
  (Company master data: Name, VAT Number, Base Currency, Logo)
  - Columns:
    -- id (UUID/Integer - Primary Key)
    -- company_name_en (String - English Name)
    -- company_name_ar (String - Arabic Name)
    -- vat_number (String - 15 digit ZATCA requirement)
    -- commercial_registration_number (String - CR Number)
    -- base_currency (String - e.g., SAR)
    -- timezone (String)
    -- logo_url (String)
    -- building_no (String - ZATCA address requirement)
    -- street_name_en (String)
    -- street_name_ar (String - ZATCA address requirement)
    -- district_en (String)
    -- district_ar (String - ZATCA address requirement)
    -- city_en (String)
    -- city_ar (String - ZATCA address requirement)
    -- postal_code (String - ZATCA address requirement)
    -- country (String - Default 'Saudi Arabia')
    -- phone_number (String)
    -- email (String)
    -- website (String)
    -- financial_year_start (Date)
    -- financial_year_end (Date)
    -- is_active (Boolean)
    -- created_at (Timestamp)
    -- updated_at (Timestamp)
- **Centrus_Core_Branches**
  (Store/Branch locations and details)
  - Columns:
    -- id (UUID/Integer - Primary Key)
    -- company_id (Foreign Key -> Centrus_Core_Companies)
    -- branch_code (String - Unique Identifier, e.g., BR-001)
    -- branch_name_en (String - English Name)
    -- branch_name_ar (String - Arabic Name)
    -- building_no (String - ZATCA requirement if different from HQ)
    -- street_name_en (String)
    -- street_name_ar (String - ZATCA requirement)
    -- district_en (String)
    -- district_ar (String - ZATCA requirement)
    -- city_en (String)
    -- city_ar (String - ZATCA requirement)
    -- postal_code (String - ZATCA requirement)
    -- phone_number (String)
    -- email (String)
    -- is_active (Boolean)
    -- created_at (Timestamp)
    -- updated_at (Timestamp)
- **Centrus_Core_Hardware_Devices**
  (Printers, Barcode Scanners, Payment Terminals mapped to specific Branches)
  - Columns:
    -- id (UUID/Integer - Primary Key)
    -- branch_id (Foreign Key -> Centrus_Core_Branches)
    -- device_type (Enum - PRINTER, SCANNER, PAYMENT_TERMINAL, CASH_DRAWER)
    -- device_name (String - e.g., "Counter 1 Epson Printer")
    -- ip_address (String - for network devices)
    -- mac_address (String)
    -- port (String/Integer - USB or Network port)
    -- is_active (Boolean)
    -- created_at (Timestamp)
    -- updated_at (Timestamp)
- **Centrus_Core_Payment_Modes**
  (Master list of payment methods: Cash, MADA, STC Pay, Credit Card)
  - Columns:
    -- id (UUID/Integer - Primary Key)
    -- code (String - Unique code, e.g., CASH, MADA, TABBY)

    -- name_ar (String)

    -- gl_account_id (Foreign Key -> Centrus_Core_Chart_Of_Accounts - default ledger)

  (Mapping table: Which payment modes are active for which branch)
  - Columns:

  - Columns:

    -- compliance_csid (String - Temporary CSID)

    -- public_key (Text)
